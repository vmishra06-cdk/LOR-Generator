# 📄 Professional LOR Generator

A lightweight Flask web application that generates formal Letters of Recommendation in PDF format for the **Bit By Bit Club**.

## 🚀 Features
- **Instant Generation:** Fill a form and get a PDF in seconds.
- **Modern UI:** Clean, responsive interface built with CSS Glassmorphism.
- **Professional Formatting:** Standard A4 layout with logo integration via ReportLab.
- **In-Memory Processing:** PDFs are generated on-the-fly without saving files to the server.

## 🛠️ Tech Stack
- **Backend:** Python / Flask
- **PDF Engine:** ReportLab
- **Frontend:** HTML5 / CSS3
